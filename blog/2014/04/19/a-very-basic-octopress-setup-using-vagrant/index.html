A Very Basic Octopress Setup Using Vagrant

Apr 19th, 2014

Setting up Octopress can be tricky, especially on Windows systems. Using a Vagrant powered virtual machine instead we are only one command away from a blogging environment ready-to-go. Run from the directory of the Vagrantfile vagrant up will create a VirtualBox Image with Ubuntu 12.04 LTS.

The actual installation is done by shell provisioning. octopress.sh has to be included in the scripts folder relative to the Vagrantfile. It installs git, ruby and grants to the default user of Vagrant images.

Only thing left is to create a posts folder which is used to access the markdown files generated by rake new_post with an IDE from the host system. Any changes on will be synchronized automatically. The complete project can be found here.
